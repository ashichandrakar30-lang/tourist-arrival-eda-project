Tourist Arrival EDA Project
Project Title

Tourist Arrival Data Analysis

Objective

Explore tourist arrival data

Find trends and patterns

Visualize using simple charts

Dataset

File: tourist_data.csv

Columns: y (total tourists), f1–f41 (monthly/other features), Year, Month, Index

Tools

Python 3

Pandas

Matplotlib

Steps

Loaded CSV file and checked data

Plotted line chart of tourist arrivals over time

Plotted histogram to see distribution of tourists

Found top correlated features with y

Plotted scatter plots of top features vs y

Plotted average trend of monthly features f1–f41

Observations

Tourist arrivals show seasonal and yearly trends

Top features have strong relation with tourist numbers

Monthly features show repeating patterns

How to Run

Install Python 3, Pandas, and Matplotlib:

pip install pandas matplotlib


Place CSV in the same folder as script

Run script:

python tourist_arrival_data_analysis.py

Conclusion

Simple EDA and charts help understand tourist arrival trends and patterns.
Charts clearly show peak months, distribution, and top features affecting tourism.
